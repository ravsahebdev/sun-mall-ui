# ☀️ sun-mall-ui

A modern snack ecommerce landing page built using HTML & CSS.

This project was created as part of Assignment 3 from Cohort 3.0 by Sheryians Coding School while practicing Flexbox, CSS positioning, ecommerce layout structuring, and premium UI composition concepts.

---

## 🚀 Features

- Modern Ecommerce UI
- Product Showcase Layout
- Premium Card Composition
- Gradient Background Effects
- Interactive Navigation
- Floating Decorative Elements
- Modern Typography
- Smooth Hover Effects

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Positioning
- Font Awesome
- Remix Icons
- Google Fonts

---

## 📚 Concepts Practiced

- display: flex
- position: relative
- position: absolute
- UI Structuring
- Product Layout Design
- Alignment & Spacing
- Layering & Visual Hierarchy
- Ecommerce Composition
